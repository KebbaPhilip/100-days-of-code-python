# Day 21 – Snake Game (Part 2)

## Project Description

This project completes the Snake Game by adding food generation, scoring,
wall collision detection, and self collision logic.

## Concepts Learned

- Collision detection
- Inheritance with Turtle
- Game state management
- Score tracking

## File Structure

- `snake.py` – Snake behavior and growth
- `food.py` – Random food generation
- `scoreboard.py` – Score tracking and game-over display
- `main.py` – Game loop and event handling

## How to Run

```bash
python main.py
```
