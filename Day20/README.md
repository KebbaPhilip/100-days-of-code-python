# Day 20 – Snake Game (Part 1)

## Project Description

This is the first part of the Snake Game project. It implements the snake
body, movement logic, and keyboard controls using Python's turtle module.

## Concepts Learned

- Object-Oriented Programming
- Game loops
- Screen animation control
- Keyboard event handling

## File Structure

- `snake.py` – Snake class and movement logic
- `main.py` – Game loop and screen setup

## How to Run

```bash
python main.py
```
