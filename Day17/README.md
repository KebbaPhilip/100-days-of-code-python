# Day 17 – Quiz Game

## Project Description

A command-line quiz game that asks True/False questions.
The project is structured across multiple files, each
handling a specific responsibility.

## File Structure

- `question_model.py` – Question class
- `quiz_brain.py` – Game logic
- `data.py` – Question data
- `main.py` – Program entry point

## Concepts Learned

- Object-Oriented Programming
- Working with multiple Python files
- Classes and instances
- Game logic with loops

## How to Run

```bash
python main.py
```
