# Day 22 – Pong Game

## Overview

This project recreates the classic Pong arcade game using Python's turtle module.  
It demonstrates object-oriented programming, collision detection, and real-time animation.

## Features

- Two-player paddle controls
- Ball movement with bounce physics
- Wall collision detection
- Paddle collision detection
- Score tracking system
- Automatic ball reset after a point

## Controls

Right Paddle:

- Up Arrow → Move Up
- Down Arrow → Move Down

Left Paddle:

- W → Move Up
- S → Move Down

## Concepts Practiced

- Object-Oriented Programming (OOP)
- Inheritance
- Game loops
- Collision detection
- Event listeners
- Screen animation control

## How to Run

Make sure Python is installed.

```bash
python main.py
```
